## What is Kosmosic?

Kosmosic is a community-driven study focus app designed to help you stay motivated and accountable. Built with JavaScript and Vite, it delivers a seamless, cross-platform productivity experience across web, desktop, and mobile.

**Website:** [kosmosic.vercel.app](https://kosmosic.vercel.app)
